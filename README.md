# How to run

1 - Clone this repo:

```
> git clone git@github.com:macabeus/js-proposal-algebraic-effects.git
```

2 - Run some commands on Babel:

```
> cd babel
> make bootstrap
> make watch
```

3 - So you could run the example with:

```
> cd babel-algebraic-effects
> yarn start
```
